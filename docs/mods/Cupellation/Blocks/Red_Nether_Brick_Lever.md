## Description
The red nether brick lever is a [lever](https://minecraft.wiki/w/Lever) which can be [switched](https://minecraft.wiki/w/Redstone_components) to provide [redstone power](https://minecraft.wiki/w/Redstone_circuits) (for a smelter to maintain heat for example).

## Obtaining
#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    5=stick, Stick, https://minecraft.wiki/w/Stick;
    8=red_nether_bricks, Red Nether Bricks, https://minecraft.wiki/w/Red_Nether_Bricks
]
output[cupellation:red_nether_brick_lever, Red Nether Brick Lever]
</div>
</div>

## Usage
The lever can be [switched](https://minecraft.wiki/w/Redstone_components) by right clicking it.
