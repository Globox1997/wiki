## Description
The deepslate brick casting table is used to cast molds or moldable items in a smelter.

## Usage
A deepslate casting table can be filled if a [faucet](/wiki/mods/Cupellation/Blocks/Deepslate_Brick_Faucet) is one or two blocks above the table and either a mold item lays on the table or an [item](https://minecraft.wiki/w/Item) like an ingot lays on the table to cast a mold. If enough material is inside the smelter, right click the faucet to start filling the mold or table.
