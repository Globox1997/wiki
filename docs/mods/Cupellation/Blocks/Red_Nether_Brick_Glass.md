## Description
The red nether brick glass is a transparent block which is used to build a smelter out of and see the molten material inside the smelter at the same time.

## Obtaining
#### Blast Furnace
<div id="crafting-table">
<div class="crafting-element" crafting-type="blast_furnace_smelting">
input[
    1=red_nether_bricks, Red Nether Bricks, https://minecraft.wiki/w/Red_Nether_Bricks;
    2=quartz, Quartz, https://minecraft.wiki/w/Quartz
]
output[cupellation:red_nether_brick_glass, Red Nether Brick Glass]
</div>
</div>

## Usage
Just build the smelter wall out of the glass block to see which molten material is inside the smelter.
