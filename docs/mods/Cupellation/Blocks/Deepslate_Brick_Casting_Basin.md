## Description
The deepslate brick casting basin is used to cast full blocks of material in a smelter.

## Usage
A deepslate casting basin can be filled if a [faucet](/wiki/mods/Cupellation/Blocks/Deepslate_Brick_Faucet) is one or two blocks above the basin. If enough material is inside the smelter, right click the faucet to start filling the basin.
