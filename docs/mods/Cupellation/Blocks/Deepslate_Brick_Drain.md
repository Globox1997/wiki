## Description
The deepslate brick drain is a block which is required in a smelter to be able to cast molten material. A [faucet](/wiki/mods/Cupellation/Blocks/Deepslate_Brick_Faucet) can be placed on its output side.

## Obtaining
#### Stonecutting
<div id="crafting-table">
<div class="crafting-element" crafting-type="stonecutting">
input[ 1=deepslate_bricks, Deepslate Bricks, https://minecraft.wiki/w/Deepslate_Bricks ]
output[cupellation:deepslate_brick_drain, Deepslate Brick Drain]
</div>
</div>

## Usage
If a [faucet](/wiki/mods/Cupellation/Blocks/Deepslate_Brick_Faucet) is placed on the drains output side, the drain will let molten material flow through itself.
