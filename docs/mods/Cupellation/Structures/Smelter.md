## Description
The smelter is a multiblock [structure](https://minecraft.wiki/w/Structure) and can smelt material when built correctly.

## Generation
The [player](https://minecraft.wiki/w/Player) has to build the [structure](https://minecraft.wiki/w/Structure) anywhere. If built correctly, the main smelter block can be accessed. The smelter has a minimum width of 5, maximum width of 7 blocks, minimum height of 3 and maximum height of 8 blocks.

#### Blocks
- 56 Deepslate Bricks
- 1 Deepslate Brick Smelter
