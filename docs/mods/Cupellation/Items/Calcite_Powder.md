## Description
The calcite powder is a flux item used to reduce slag in the smelter.

## Obtaining
#### Stonecutting
<div id="crafting-table">
<div class="crafting-element" crafting-type="stonecutting">
input[ 1=calcite, Calcite, https://minecraft.wiki/w/Calcite ]
output[cupellation:calcite_powder, Calcite Powder]
</div>
</div>

## Usage
Calcite Powder can be thrown inside the smelter to reduce specific kinds of slag.
