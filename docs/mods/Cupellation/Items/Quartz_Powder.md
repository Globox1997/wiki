## Description
The quartz powder is a flux item used to reduce slag in the smelter.

## Obtaining
#### Stonecutting
<div id="crafting-table">
<div class="crafting-element" crafting-type="stonecutting">
input[ 1=quartz, Nether Quartz, https://minecraft.wiki/w/Nether_Quartz ]
output[cupellation:quartz_powder, Quartz Powder]
</div>
</div>

## Usage
Quartz powder can be thrown inside the smelter to reduce specific kinds of slag.
