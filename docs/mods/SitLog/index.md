# Description
SitLog adds the ability to sit down anywhere. When the [player](https://minecraft.wiki/w/Player) disconnects, the [players](https://minecraft.wiki/w/Player) body will not despawn and can still be attacked, this prevents combat logging completely.
If you want to you can configure how long the [players](https://minecraft.wiki/w/Player) body will stay.
