## Description
A steel horse armor is a mid game [horse armor](https://minecraft.wiki/w/Horse_Armor). Unlike other steel items, it cannot be [crafted](https://minecraft.wiki/w/Crafting) and is only obtainable from [chest loot](https://minecraft.wiki/w/Generated_loot) in various [structures](https://minecraft.wiki/w/Structure). 

## Usage
A steel horse armor can be equiped on a regular [horse](https://minecraft.wiki/w/Horse) by placing it in the respective horse inventory slot.
