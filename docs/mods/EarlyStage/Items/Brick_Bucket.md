## Description
A brick bucket is a [bucket](https://minecraft.wiki/w/Bucket) item. It can be filled with [water](https://minecraft.wiki/w/Water) and [lava](https://minecraft.wiki/w/Lava).

## Usage
Empty brick buckets can be used to "pick up" a [water](https://minecraft.wiki/w/Water) or [lava](https://minecraft.wiki/w/Lava) source block by pressing use [item](https://minecraft.wiki/w/Item) on a [block](https://minecraft.wiki/w/Block) of the relevant type. If the empty brick bucket is part of a stack and the player's [inventory](https://minecraft.wiki/w/Inventory) is full, the filled bucket drops in front of the [player](https://minecraft.wiki/w/Player) as an [item](https://minecraft.wiki/w/Item).
