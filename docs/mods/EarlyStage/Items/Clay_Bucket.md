## Description
A clay bucket is an [item](https://minecraft.wiki/w/Item) which can not contain a fluid.

## Obtaining
#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    4=clay_ball, Clay Ball, https://minecraft.wiki/w/Clay_Ball;
    6=clay_ball, Clay Ball, https://minecraft.wiki/w/Clay_Ball;
    8=clay_ball, Clay Ball, https://minecraft.wiki/w/Clay_Ball
]
output[earlystage:clay_bucket, Clay Bucket ]
</div>
</div>

## Usage
If a clay bucket is used on a [lava](https://minecraft.wiki/w/Lava) source it will turn to a [brick bucket](/wiki/mods/EarlyStage/Items/Brick_Bucket) which can pick up fluids.
<div id="crafting-table">
<div class="crafting-element" crafting-type="furnace_smelting">
input[
    1=earlystage:clay_bucket, Clay Bucket, EarlyStage/Items/Clay_Bucket, true
]
output[earlystage:brick_bucket, Brick Bucket ]
</div>
</div>
