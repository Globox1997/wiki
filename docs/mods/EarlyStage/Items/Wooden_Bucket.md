## Description
A wooden bucket is a [bucket](https://minecraft.wiki/w/Bucket) item. It can be filled with [water](https://minecraft.wiki/w/Water). It can be used only once before it breaks.

## Usage
Empty wooden buckets can be used to "pick up" a [water](https://minecraft.wiki/w/Water) source block by pressing use [item](https://minecraft.wiki/w/Item) on a [water](https://minecraft.wiki/w/Water) source. If the empty wooden bucket is part of a stack and the player's [inventory](https://minecraft.wiki/w/Inventory) is full, the filled wooden bucket drops in front of the [player](https://minecraft.wiki/w/Player) as an [item](https://minecraft.wiki/w/Item).
