## Description
The crimson bark is an [item](https://minecraft.wiki/w/Item) used for [fuel](https://minecraft.wiki/w/Smelting#Fuel) to [smelt](https://minecraft.wiki/w/Smelting) [items](https://minecraft.wiki/w/Item) and can be used light up a [campfire](https://minecraft.wiki/w/Campfire).

## Obtaining
#### Harvesting
If an [axe](https://minecraft.wiki/w/Axe) is used on a [log](https://minecraft.wiki/w/Log) it will drop its bark.

## Usage
A [stripped log](https://minecraft.wiki/w/Stripped_Log) can be turned back into its original [log](https://minecraft.wiki/w/Log) variant by putting the bark back on it. Most importantly the bark is used to light up a [campfire](https://minecraft.wiki/w/Campfire) by using the bark on an unlit [campfire](https://minecraft.wiki/w/Campfire).
