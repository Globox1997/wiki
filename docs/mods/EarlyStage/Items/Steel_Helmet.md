## Description
A steel helmet is a mid game [armor](https://minecraft.wiki/w/Armor) piece that provides [players](https://minecraft.wiki/w/Player) and certain [mobs](https://minecraft.wiki/w/Mob) protection from common [damage types](https://minecraft.wiki/w/Damage_type) and appear graphically on the wearer.

## Usage
A steel helmet can be worn in the head slot of the [inventory](https://minecraft.wiki/w/Inventory) to give <span class="icon-element" icon-count="3" icon-id="armor"></span> [protection](https://minecraft.wiki/w/Armor#Protection).
