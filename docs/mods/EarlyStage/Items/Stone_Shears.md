## Description
Stone shears is an early game tool similar to [shears](https://minecraft.wiki/w/Shears) and is used to obtain some organic blocks like [wool](https://minecraft.wiki/w/Wool).

## Obtaining
#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    5=cobblestone, Stone Materials, https://minecraft.wiki/w/Cobblestone;
    7=cobblestone, Stone Materials, https://minecraft.wiki/w/Cobblestone
]
output[earlystage:stone_shears, Stone Shears ]
</div>
</div>

## Usage
Use stone shears as a normal [tool](https://minecraft.wiki/w/Tool) or right click a [sheep](https://minecraft.wiki/w/Sheep) to remove its coat which will drop the corresponding color [wool](https://minecraft.wiki/w/Wool).
