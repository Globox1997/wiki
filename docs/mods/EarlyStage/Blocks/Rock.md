## Description
A rock will generate ontop of [stone](https://minecraft.wiki/w/Stone) and [dirt](https://minecraft.wiki/w/Dirt) like [blocks](https://minecraft.wiki/w/Block) inside multiple different biomes, like forest biomes, hill biomes, mountain biomes and river biomes. It is a source for crafting the [crafting rock](/wiki/mods/EarlyStage/Blocks/Crafting_Rock) and the tool to process the recipes on the [crafting rock](/wiki/mods/EarlyStage/Blocks/Crafting_Rock).

## Obtaining
#### Breaking
If a rock is broken it will drop itself.

## Usage
Rocks can be used to process the recipes on the [crafting rock](/wiki/mods/EarlyStage/Blocks/Crafting_Rock) by holding it in the hand and right click the [crafting rock](/wiki/mods/EarlyStage/Blocks/Crafting_Rock).
