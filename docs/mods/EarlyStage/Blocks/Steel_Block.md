## Description
A block of steel, known internally as an steel block, is a precious [metal](https://minecraft.wiki/w/Metal) block equivalent to nine [steel ingots](/wiki/mods/EarlyStage/Items/Steel_Ingot). 

## Obtaining
#### Breaking
Blocks of steel can be mined only with a [stone pickaxe](https://minecraft.wiki/w/Stone_Pickaxe) or better. If a block of steel is mined without the use of a pickaxe, it drops nothing. 

## Usage
Blocks of steel can be used to store [steel ingots](/wiki/mods/EarlyStage/Items/Steel_Ingot) in a compact fashion.
