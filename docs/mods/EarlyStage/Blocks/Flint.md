## Description
The flint block will generate ontop of [stone](https://minecraft.wiki/w/Stone) and [dirt](https://minecraft.wiki/w/Dirt) like [blocks](https://minecraft.wiki/w/Block) inside multiple different biomes, like forest biomes, hill biomes, mountain biomes and river biomes. It is a source for [flint](https://minecraft.wiki/w/Flint) to craft flint tools.

## Obtaining
#### Breaking
If the flint block is broken it will drop 1 [flint](https://minecraft.wiki/w/Flint).

## Usage
[Flint](https://minecraft.wiki/w/Flint) can be used to craft flint tools or [arrows](https://minecraft.wiki/w/Arrows).
