## Description
The sieve is an utility [block](https://minecraft.wiki/w/Block) which is used for sift blocks. It can sieve blocks like [dirt](https://minecraft.wiki/w/Dirt) [blocks](https://minecraft.wiki/w/Block), [gravel](https://minecraft.wiki/w/Gravel), [moss](https://minecraft.wiki/w/Moss), [sand](https://minecraft.wiki/w/Sand) [blocks](https://minecraft.wiki/w/Block) and [clay](https://minecraft.wiki/w/Clay).
