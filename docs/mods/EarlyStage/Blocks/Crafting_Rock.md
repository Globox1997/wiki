## Description
The crafting rock is an utility [block](https://minecraft.wiki/w/Block) somewhat similar to the [crafting table](https://minecraft.wiki/w/Crafting_Table). There is no graphical user interface to open up but a visible grid on the block itself.

## Obtaining
#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    4=earlystage:rock, Rock;
    5=earlystage:rock, Rock
]
output[earlystage:crafting_rock, Crafting Rock]
</div>
</div>

## Usage
Place [items](https://minecraft.wiki/w/Item) on the grid with the same shape as you would inside the crafting gui of the [crafting table](https://minecraft.wiki/w/Crafting_Table) and use a [rock]() with right click on it (to basically hit it) and do that multiple times to [craft](https://minecraft.wiki/w/Crafting) an [item](https://minecraft.wiki/w/Item). You will need an empty hand to get the [items](https://minecraft.wiki/w/Item) back from the grid.
