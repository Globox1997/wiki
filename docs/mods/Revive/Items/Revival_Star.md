## Description
The revival star is a brewing recipe item, used to obtain the [supportive revivify potion](/wiki/mods/Revive/Items/Supportive_Revivify_Potion/).

## Obtaining
#### Trading
The revival star is a trade of the [cleric villager](https://minecraft.wiki/w/Cleric) which requires [emeralds](https://minecraft.wiki/w/Emerald) for the trade.

#### Brewing
