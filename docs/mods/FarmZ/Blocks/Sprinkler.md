## Description
A sprinkler is used to moisture [farmland](https://minecraft.wiki/w/Farmland). It requires [water](https://minecraft.wiki/w/Water) and a [redstone signal](https://minecraft.wiki/w/Redstone_circuits#Power) to work. It has a radius of 4 [blocks](https://minecraft.wiki/w/Block) to moisture [farmland](https://minecraft.wiki/w/Farmland).

## Usage
Sprinklers just need [redstone power](https://minecraft.wiki/w/Redstone_circuits#Power) and need to get filled up with a [water bucket](https://minecraft.wiki/w/Water_Bucket) to start watering the nearby [farmland](https://minecraft.wiki/w/Farmland) blocks.
