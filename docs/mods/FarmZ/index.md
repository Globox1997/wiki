# Description
This mod adds farming features like golden crops which can be grown while wearing a farmers hat. There is a certain chance when wearing this hat that the crop will be golden to obtain high quality food.

By default the mod reduces the range of water moisture range to 0!
This can be changed via the config.
To moisture farmland you will either need to use the provided water cans or a sprinkler.
