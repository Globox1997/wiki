# Description
Enhances the advancement system of minecraft and reuses it for a quest system.
This is an old mod idea, started a couple years ago and now I finished it.
It is meant for a simple main quest line.
With the provided ingame editor you can create quests.
Just activate the editor mode at the top right corner of the quest screen.

If editor mode is activated (requires op + creative mode):

- Left click + hold to drag
- Middle click to copy and edit
- Right click to edit

---
## Content
This mod does not add blocks, items nor entities.  
Check out the ingame quest screen (default keybind: l).
