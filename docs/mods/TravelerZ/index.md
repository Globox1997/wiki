# Description
TravelerZ is an addon for the mod [Nameplate](/wiki/mods/Nameplate/).  
It adds titles on traveling through [biomes](https://minecraft.wiki/w/Biome) including the average level of [mobs](https://minecraft.wiki/w/Mob).
