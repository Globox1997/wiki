## Description
A block of solarite, known internally as a solarite block, is a precious [metal](https://minecraft.wiki/w/Metal) block equivalent to nine [solarite ingots](/wiki/mods/Oblivion/Items/Solarite_Ingot). 

## Obtaining
#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    1=oblivion:solarite_ingot, Solarite Ingot;
    2=oblivion:solarite_ingot, Solarite Ingot;
    3=oblivion:solarite_ingot, Solarite Ingot;
    4=oblivion:solarite_ingot, Solarite Ingot;
    5=oblivion:solarite_ingot, Solarite Ingot;
    6=oblivion:solarite_ingot, Solarite Ingot;
    7=oblivion:solarite_ingot, Solarite Ingot;
    8=oblivion:solarite_ingot, Solarite Ingot;
    9=oblivion:solarite_ingot, Solarite Ingot
]
output[oblivion:solarite_block, Block of Solarite ]
</div>
</div>

#### Breaking
Blocks of solarite can be mined only with a [solarite pickaxe](/wiki/mods/Oblivion/Items/Solarite_Pickaxe) or better. If a block of solarite is mined without the use of a pickaxe, it drops nothing. 

## Usage
Blocks of solarite can be used to store [solarite ingots](/wiki/mods/Oblivion/Items/Solarite_Ingot) in a compact fashion.
