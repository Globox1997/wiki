## Description
A block of raw solarite, known internally as a raw solarite block, is a precious [metal](https://minecraft.wiki/w/Metal) block equivalent to nine [raw solarite](/wiki/mods/Oblivion/Items/Raw_Solarite). 

## Obtaining
#### Breaking
Blocks of raw solarite can be mined only with a [solarite pickaxe](/wiki/mods/Oblivion/Items/Solarite_Pickaxe) or better. If a block of raw solarite is mined without the use of a pickaxe, it drops nothing. A [drill](/wiki/mods/Oblivion/Blocks/Drill) can be used to mine it and will output multiple [raw solarite](/wiki/mods/Oblivion/Items/Raw_Solarite).

#### Natural Generation
Blocks of raw solarite can be found within the Oblivion Dimension in ocean biomes.

## Usage
Blocks of raw solarite can be used to store [raw solarite](/wiki/mods/Oblivion/Items/Raw_Solarite) in a compact fashion.
