## Description
Shimmering grass is a non solid plant block that may drop [wheat seeds](https://minecraft.wiki/w/Wheat_Seeds) when broken and will emit light when an [entity](https://minecraft.wiki/w/Entity) touches it. It can only be found inside the Oblivion dimension.
