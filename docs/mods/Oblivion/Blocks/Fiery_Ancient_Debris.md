## Description
Fiery ancient debris is a rare ore and the main source of [raw fiery netherite](/wiki/mods/Oblivion/Items/Raw_Fiery_Netherite).

## Obtaining
#### Breaking
Blocks of fiery ancient debris can be mined only with a [fiery netherite pickaxe](/wiki/mods/Oblivion/Items/Fiery_Netherite_Pickaxe) or better. If a fiery ancient debris is mined without the use of a pickaxe, it drops nothing. A [drill](/wiki/mods/Oblivion/Blocks/Drill) can be used to mine it and will output multiple [raw fiery netherite](/wiki/mods/Oblivion/Items/Raw_Fiery_Netherite).

#### Natural Generation
Blocks of fiery ancient debris can be found within the Oblivion Dimension in forest biomes.
