## Description
A block of raw scarlet, known internally as a raw scarlet block, is a precious [metal](https://minecraft.wiki/w/Metal) block equivalent to nine [raw scarlet](/wiki/mods/Oblivion/Items/Raw_Scarlet). 

## Obtaining
#### Breaking
Blocks of raw scarlet can be mined only with a [scarlet pickaxe](/wiki/mods/Oblivion/Items/Scarlet_Pickaxe) or better. If a block of raw scarlet is mined without the use of a pickaxe, it drops nothing. A [drill](/wiki/mods/Oblivion/Blocks/Drill) can be used to mine it and will output multiple [raw scarlet](/wiki/mods/Oblivion/Items/Raw_Scarlet).

#### Natural Generation
Blocks of raw scarlet can be found within the Oblivion Dimension in plains, meadow and silver forest biomes.
