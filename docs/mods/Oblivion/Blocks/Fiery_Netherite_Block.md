## Description
A block of fiery netherite, known internally as a fiery netherite block, is a precious [metal](https://minecraft.wiki/w/Metal) block equivalent to nine [fiery netherite ingots](/wiki/mods/Oblivion/Items/Fiery_Netherite_Ingot). 

## Obtaining
#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    1=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot;
    2=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot;
    3=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot;
    4=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot;
    5=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot;
    6=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot;
    7=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot;
    8=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot;
    9=oblivion:fiery_netherite_ingot, Fiery Netherite Ingot
]
output[oblivion:fiery_netherite_block, Block of Fiery Netherite ]
</div>
</div>

#### Breaking
Blocks of fiery netherite can be mined only with a [fiery netherite pickaxe](/wiki/mods/Oblivion/Items/Fiery_Netherite_Pickaxe) or better. If a block of fiery netherite is mined without the use of a pickaxe, it drops nothing.

## Usage
Blocks of fiery netherite can be used to store [fiery netherite ingots](/wiki/mods/Oblivion/Items/Fiery_Netherite_ingot) in a compact fashion.
