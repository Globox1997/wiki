## Description
A block of scarlet, known internally as a scarlet block, is a precious [metal](https://minecraft.wiki/w/Metal) block equivalent to nine [scarlet ingots](/wiki/mods/Oblivion/Items/Scarlet_Ingot). 

## Obtaining
#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    1=oblivion:scarlet_ingot, Scarlet Ingot;
    2=oblivion:scarlet_ingot, Scarlet Ingot;
    3=oblivion:scarlet_ingot, Scarlet Ingot;
    4=oblivion:scarlet_ingot, Scarlet Ingot;
    5=oblivion:scarlet_ingot, Scarlet Ingot;
    6=oblivion:scarlet_ingot, Scarlet Ingot;
    7=oblivion:scarlet_ingot, Scarlet Ingot;
    8=oblivion:scarlet_ingot, Scarlet Ingot;
    9=oblivion:scarlet_ingot, Scarlet Ingot
]
output[oblivion:scarlet_block, Block of Scarlet ]
</div>
</div>

#### Breaking
Blocks of scarlet can be mined only with a [scarlet pickaxe](/wiki/mods/Oblivion/Items/Scarlet_Pickaxe) or better. If a block of scarlet is mined without the use of a pickaxe, it drops nothing. 

## Usage
Blocks of scarlet can be used to store [scarlet ingots](/wiki/mods/Oblivion/Items/Scarlet_Ingot) in a compact fashion.
