## Description
Quicksand is a block found in the Oblivion dimension in forest and silver forest biomes. Standing on a quicksand block will cause the [player](https://minecraft.wiki/w/Player) to sink, resulting in block suffocation unless the quicksand block is broken.

## Obtaining
#### Breaking
Quicksand can be mined with a [shovel](https://minecraft.wiki/w/Shovel).
