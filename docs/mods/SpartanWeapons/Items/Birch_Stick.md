## Description
The birch stick is an [item](https://minecraft.wiki/w/Item) used to craft several weapons.

## Obtaining
#### Loot
The birch stick is dropped by [birch leaves](https://minecraft.wiki/w/Birch_Leaves).

#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    5=birch_planks, Birch Planks, https://minecraft.wiki/w/Birch_Planks;
    8=birch_planks, Birch Planks, https://minecraft.wiki/w/Birch_Planks
]
output[spartanweapons:birch_stick, Birch Stick ]
</div>
</div>

## Usage
The birch stick can be used to craft different weapons just like with the vanilla [stick](https://minecraft.wiki/w/Stick).
