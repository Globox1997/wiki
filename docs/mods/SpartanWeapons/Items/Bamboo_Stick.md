## Description
The bamboo stick is an [item](https://minecraft.wiki/w/Item) used to craft several weapons.

## Obtaining
#### Loot
The bamboo stick is dropped by [bamboo leaves](https://minecraft.wiki/w/Bamboo_Leaves).

#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    5=bamboo_planks, Bamboo Planks, https://minecraft.wiki/w/Bamboo_Planks;
    8=bamboo_planks, Bamboo Planks, https://minecraft.wiki/w/Bamboo_Planks
]
output[spartanweapons:bamboo_stick, Bamboo Stick ]
</div>

## Usage
The bamboo stick can be used to craft different weapons just like with the vanilla [stick](https://minecraft.wiki/w/Stick).
