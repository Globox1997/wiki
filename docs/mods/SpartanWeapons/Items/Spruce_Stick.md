## Description
The spruce stick is an [item](https://minecraft.wiki/w/Item) used to craft several weapons.

## Obtaining
#### Loot
The spruce stick is dropped by [spruce leaves](https://minecraft.wiki/w/spruce_Leaves).

#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    5=spruce_planks, Spruce Planks, https://minecraft.wiki/w/Spruce_Planks;
    8=spruce_planks, Spruce Planks, https://minecraft.wiki/w/Spruce_Planks
]
output[spartanweapons:spruce_stick, Spruce Stick ]
</div>
</div>

## Usage
The spruce stick can be used to craft different weapons just like with the vanilla [stick](https://minecraft.wiki/w/Stick).
