## Description
The cherry stick is an [item](https://minecraft.wiki/w/Item) used to craft several weapons.

## Obtaining
#### Loot
The cherry stick is dropped by [cherry leaves](https://minecraft.wiki/w/cherry_Leaves).

#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">

    8=cherry_planks, Cherry Planks, https://minecraft.wiki/w/Cherry_Planks
]
output[spartanweapons:cherry_stick, Cherry Stick ]

## Usage
The cherry stick can be used to craft different weapons just like with the vanilla [stick](https://minecraft.wiki/w/Stick).
