## Description
The mangrove stick is an [item](https://minecraft.wiki/w/Item) used to craft several weapons.

## Obtaining
#### Loot
The mangrove stick is dropped by [mangrove leaves](https://minecraft.wiki/w/mangrove_Leaves).

#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    5=mangrove_planks, Mangrove Planks, https://minecraft.wiki/w/Mangrove_Planks;
    8=mangrove_planks, Mangrove Planks, https://minecraft.wiki/w/Mangrove_Planks
]
output[spartanweapons:mangrove_stick, Mangrove Stick ]
</div>
</div>

## Usage
The mangrove stick can be used to craft different weapons just like with the vanilla [stick](https://minecraft.wiki/w/Stick).
