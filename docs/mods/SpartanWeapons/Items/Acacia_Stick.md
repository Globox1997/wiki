## Description
The acacia stick is an [item](https://minecraft.wiki/w/Item) used to craft several weapons.

## Obtaining
#### Loot
The acacia stick is dropped by [acacia leaves](https://minecraft.wiki/w/Acacia_Leaves).

#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">

    5=acacia_planks, Acacia Planks, https://minecraft.wiki/w/Acacia_Planks;
    8=acacia_planks, Acacia Planks, https://minecraft.wiki/w/Acacia_Planks

## Usage
The acacia stick can be used to craft different weapons just like with the vanilla [stick](https://minecraft.wiki/w/Stick).
