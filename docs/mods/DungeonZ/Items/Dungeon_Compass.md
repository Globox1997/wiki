## Description
The dungeon compass is an [item](https://minecraft.wiki/w/Item) really similar to a normal [compass](https://minecraft.wiki/w/Compass) but if calibrated it will point to the calibrated dungeon entrance. It can be calibrated at a [cartography table](https://minecraft.wiki/w/Cartography_Table) by using it on the table.

## Usage
While holding the dungeon compass use right click on a cartography table to open up the calibration screen.
