## Description
The temple dungeon [structure](https://minecraft.wiki/w/Structure) is the entrance to go to the temple dungeon. If found, it requires specific [items](https://minecraft.wiki/w/Item) (depending on the set difficulty of the dungeon) to enter the dungeon.
Usually it requires [diamonds](https://minecraft.wiki/w/Diamond) to enter it but the [loot](https://minecraft.wiki/w/Drops) granted might be worth it. But watch out this dungeon has its difficulties.

## Generation
The temple dungeon entrance can be found anywhere in the [overworld](https://minecraft.wiki/w/Overworld) but it is quite rare. Use a [dungeon compass](/wiki/mods/DungeonZ/Items/Dungeon_Compass) to find it.

The entrance generates in the following biomes:
- [plains](https://minecraft.wiki/w/Plains)
- [plains](https://minecraft.wiki/w/Forest)
- [plains](https://minecraft.wiki/w/Jungle)
- [plains](https://minecraft.wiki/w/Dark_Forest)
- [plains](https://minecraft.wiki/w/Taiga)
- [plains](https://minecraft.wiki/w/Birch_Forest)
