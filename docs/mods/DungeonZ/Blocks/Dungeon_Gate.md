## Description
The dungeon gate block is a [block](https://minecraft.wiki/w/Block) generated inside a dungeon. It will block curtain paths of a dungeon until a criteria is met by the player.

## Obtaining
The dungeon Gate block cannot be obtained as an [item](https://minecraft.wiki/w/Item) only by using the /give [command](https://minecraft.wiki/w/Commands), in [Creative mode](https://minecraft.wiki/w/Creative) by taking it from Creative inventory or by using pick block

## Usage
Depending on how the gate settings are set, the gate will disappear when curtain requirements are met or if an unlock item is used on it. If a dungeon is build in [creative mode](https://minecraft.wiki/w/Creative) with op rights, the block can be placed and a settings screen can be opened while sneaking with an empty [hand](https://minecraft.wiki/w/Hand). To change the appearance of the gate a [block](https://minecraft.wiki/w/Block) has to get used on it.
