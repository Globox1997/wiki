## Description 
Rotten Leather is an [item](https://minecraft.wiki/w/Item) used to get processed to [leather](https://minecraft.wiki/w/Leather) which then can be crafted into other [items](https://minecraft.wiki/w/Item).

## Obtaining
#### Crafting
<div id="crafting-table">
<div class="crafting-element" crafting-type="vanilla_crafting">
input[
    4=rotten_flesh, Rotten Flesh, https://minecraft.wiki/w/Rotten_Flesh;
    5=rotten_flesh, Rotten Flesh, https://minecraft.wiki/w/Rotten_Flesh
]
output[fleshz:rotten_leather, rotten leather ]
</div>
</div>
