## Description
Hide is an [item](https://minecraft.wiki/w/Item) used to get processed to [leather](https://minecraft.wiki/w/Leather) which then can be crafted into other [items](https://minecraft.wiki/w/Item).

## Obtaining
#### Looting
Hide is typically dropped by [mobs](https://minecraft.wiki/w/Mob) like [cows](https://minecraft.wiki/w/Cow), [horses](https://minecraft.wiki/w/Horse) and other [leather](https://minecraft.wiki/w/Leather) dropping [animals](https://minecraft.wiki/w/Animal).

#### Wood Rack
<div id="crafting-table">
<div class="crafting-element" crafting-type="wood_rack">

    1=fleshz:rotten_leather, Rotten Leather
]
output[fleshz:hide, Hide]
