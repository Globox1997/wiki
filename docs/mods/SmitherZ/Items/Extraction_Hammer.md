## Description
The extraction hammer is an [item](https://minecraft.wiki/w/Item) used to increase the chance to unlink a gem from an [item](https://minecraft.wiki/w/Item). It can only be used at the [grindstone](https://minecraft.wiki/w/Grindstone).

## Obtaining
#### Loot
The extraction hammer can not be crafted but found in [buried treasure chests](https://minecraft.wiki/w/Buried_Treasure). The highest quality of the hammer can be found very rarely inside [end city](https://minecraft.wiki/w/End_City) treasure chests.

## Usage
The extraction hammer increases the total gem unlink chance by 5% up to 50% depending on the quality of the hammer. Each hammer can only be used one time. It is automatically used during the unlinking process when put inside the hammer slot at the [grindstone](https://minecraft.wiki/w/Grindstone).
