## Description
The smither hammer is an [item](https://minecraft.wiki/w/Item) used to increase the chance to link a gem on an [item](https://minecraft.wiki/w/Item). It can only be used at the [smithing table](https://minecraft.wiki/w/Smithing_Table).

## Obtaining
#### Loot
The smither hammer can not be crafted but found in [bastion](https://minecraft.wiki/w/Bastion_Remnant) treasure chests. The highest quality of the hammer can be found very rarely inside [end city](https://minecraft.wiki/w/End_City) treasure chests.

## Usage
The smither hammer increases the total gem link chance by 5% up to 50% depending on the quality of the hammer. Each hammer can only be used one time. It is automatically used during the linking process when put inside the hammer slot at the [smithing table](https://minecraft.wiki/w/Smithing_Table).
