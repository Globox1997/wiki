## Description
Warming is a [status effect](https://minecraft.wiki/w/Effect) that will reduce the [freezing temperature effect](/wiki/mods/EnvironmentZ/Mechanic/) of the environment. It can be removed by drinking [milk](https://minecraft.wiki/w/Milk_Bucket). The [overheating resistance potion](/wiki/mods/EnvironmentZ/Items/Overheating_Resistance_Potion/) will grant this status effect.

## Effect
The warming effect will reduce or even completely eliminate the freezing environment [temperature](/wiki/mods/EnvironmentZ/Mechanic/), keeping the player in a quite hot and well state.
