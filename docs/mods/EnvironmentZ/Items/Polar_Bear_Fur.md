## Description
Polar bear fur is an [item](https://minecraft.wiki/w/Item) used to insulate [armor](https://minecraft.wiki/w/Armor).

## Obtaining
#### Loot
[Polar bears](https://minecraft.wiki/w/Polar_Bear) can drop 0 - 2 polar bear fur upon death.

## Usage
Any [armor](https://minecraft.wiki/w/Armor) [item](https://minecraft.wiki/w/Item) can be insulated at the [anvil](https://minecraft.wiki/w/Anvil). To remove the insulation, put the insulated [item](https://minecraft.wiki/w/Item) and [shears](https://minecraft.wiki/w/Shears) in the [anvil](https://minecraft.wiki/w/Anvil).
