## Description
Wanderer leggings is an [armor](https://minecraft.wiki/w/Armor) piece that provides [players](https://minecraft.wiki/w/Player) and certain [mobs](https://minecraft.wiki/w/Mob) protection from common [damage types](https://minecraft.wiki/w/Damage_type) and appear graphically on the wearer.

## Usage
Wanderer leggings can be worn in the legs slot of the [inventory](https://minecraft.wiki/w/Inventory) to give <span class="icon-element" icon-count="1" icon-id="armor"></span> [protection](https://minecraft.wiki/w/Armor#Protection). It will protect the [player](https://minecraft.wiki/w/Player) from hot [temperatures](/wiki/mods/EnvironmentZ/Mechanic/).
