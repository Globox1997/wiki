## Description
The overheating resistance potion is a [consumable drink](https://minecraft.wiki/w/Drinks) [item](https://minecraft.wiki/w/Item) which grants the [cooling effect](/wiki/mods/EnvironmentZ/Effects/Cooling/) which will protect the [player](https://minecraft.wiki/w/Player) from hot [temperatures](/wiki/mods/EnvironmentZ/Mechanic/).

## Usage
To drink an overheating resistance potion, press and hold right click while it is selected in the hotbar.
