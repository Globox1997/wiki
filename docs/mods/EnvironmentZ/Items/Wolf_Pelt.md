## Description
Wolf pelt is an [item](https://minecraft.wiki/w/Item) used to craft wolf armor which protects the [player](https://minecraft.wiki/w/Player) from cold [temperatures](/wiki/mods/EnvironmentZ/Mechanic/).

## Obtaining
#### Loot
[Wolves](https://minecraft.wiki/w/Wolf) can drop 0 - 3 wolf pelt upon death.
