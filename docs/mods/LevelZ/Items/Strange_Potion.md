## Description
A strange potion is a similar [item](https://minecraft.wiki/w/Item) like a regular [potion](https://minecraft.wiki/w/Potion). It is a consumable liquid but when drunk, it will reset one random skill stat (which has at least one used level point) while granting the used level points.

## Usage
The strange potion is used by pressing and holding use while holding the potion in the mainhand.
